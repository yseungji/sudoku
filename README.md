스도쿠 풀이를 위해서는 두 가지 방법이 생각난다.

첫 번째 방법은 브루트포스로 단순 노가다를 하는 것이다. 
칸 수에 따라 최대 9^칸수 만큼 걸릴것이다. 9^10 승만 해도 3,486,784,401(약 34억)이라는 수가 나오니 시간이 너무 오래 걸릴것이다.

그래서 두 번째 방법으로 백트랙킹이 있다. 
이 방법으로 구현하면서 조심해야 할 점은 백트랙킹 하는 과정에서 이 전의 수를 0으로 다시 초기화 시켜줌으로써 불가능한 경우를 완전히 이전 상태로 되돌리는 것이 필요하다.
백트랙킹은 최적화 과정을 거치기 떄문에 시간복잡도를 많이 줄일 수 있다.
